# SelectableRichText Example

This example runs `@boomsi/react-native-selectable-text` with React Native `0.85.0` and New Architecture/Fabric enabled.

## Commands

```sh
yarn start
yarn android
yarn ios
yarn build:android
yarn build:ios
```

For iOS, install pods after native changes:

```sh
cd ios && pod install
```

The example no longer documents or validates the old Paper architecture.
